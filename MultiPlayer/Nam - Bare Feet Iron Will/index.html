    Nam - Bare Feet Iron Will


    In this war, there is no victor. You will be fighting with numerous conspiracies with soldiers who don’t have a choice. Jump back into the Vietnam war and fight the enemies you encounter on the map. You have the option of inviting your friends and start playing together. Pick your side and start fighting. The game is offering tons of fun, so enjoy it! You won’t have time to rest.



    Game Controls:
    "W,A,S,D" = Movement

    "1, 2, 3, 4" = Change weapon

    "R" = Reload

    "Mouse scroll(Middle)"" = Shooting

    "SHIFT" = Run

    "SPACE" = Jump

    "TAB" = Score

    "ESE" = Pause

    "V" = Knife

    "G" = Granade

    "T || Y" = Chat

    "M" = change Class

    "L" = Map
